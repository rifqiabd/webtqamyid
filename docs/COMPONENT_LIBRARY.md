# Component Library

## Primitives (ui/)
- Button: primary, secondary, outline, ghost, link
- Card: default, feature, ghost
- Badge: green, gold, neutral
- Tag: inline label
- Input: text, email, tel, search

## Layout (layout/)
- Header: sticky, blur, responsive hamburger
- Footer: dark teal, 4-column, social icons
- Navigation: desktop + mobile menu

## Sections (sections/)
- HeroSection (dark gradient)
- ValuesSection (4-card grid)
- EducationOverviewSection
- ProgramsOverviewSection
- StatsSection
- CTASection
- ContactInfoSection
- PageHeroSection (small dark hero for subpages)

## Content (content/)
- NewsCard
- ProgramCard
- GalleryItem

## Conventions
- Every component uses interface Props
- Supports class: passthrough
- No exports (single .astro file)
- data- attributes for testing
