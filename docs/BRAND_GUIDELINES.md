# Brand Guidelines — Pondok Pesantren Thibbil Qulub Assimbani

## Brand Personality
- Trustworthy, calm, intelligent, modern, Islamic, educational, timeless, professional

## Design Influences
- **50% Notion**: Warm neutrals, generous body leading, content-first, borders over shadows
- **30% Vercel**: Monochrome precision, aggressive negative letter-spacing, generous section spacing  
- **20% MongoDB**: Dual-mode dark/light, accent reserved for CTAs, dark footer

## Color Palette
- **Primary**: Deep Islamic Green (#1a7a4c) — CTAs, primary actions
- **Accent**: Warm Gold (#c8a45c) — highlights, sacred elements
- **Neutrals**: Notion-inspired warm scale (#37352f text, #f6f5f4 surfaces)

## Typography
- Inter (humanist geometric) for all UI
- JetBrains Mono for code blocks
- Display: negative tracking, tight leading
- Body: generous 1.55 leading

## Tone of Voice
- Warm, welcoming, professional, faith-centered
- Indonesian language, clear and direct
- No corporate buzzwords, no AI-sounding copy
