# Design System — Pondok Pesantren Thibbil Qulub Assimbani

## Spacing
- Base unit: 4px
- Scale: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 96, 120, 192
- Section padding: 96px (default), 64px (dense), 128px (hero)

## Typography Scale
| Token | Size | LineH | Tracking | Use |
|---|---|---|---|---|
| display-xl | 4rem | 1.05 | -0.02em | Hero headline |
| display-lg | 3rem | 1.10 | -0.01em | Section openers |
| heading-1 | 2.5rem | 1.15 | -0.005em | Page H1 |
| heading-2 | 2rem | 1.20 | -0.005em | Section H2 |
| heading-3 | 1.5rem | 1.25 | 0 | Card titles |
| body | 1rem | 1.55 | 0 | Body |
| body-sm | 0.875rem | 1.50 | 0 | Secondary |
| caption | 0.8125rem | 1.40 | 0 | Labels |

## Border Radius
- Buttons: 8px (Notion principle — rectangles, not pills)
- Cards: 12px
- PPDB CTA: pill-shaped (MongoDB principle)
- Badges/tags: full rounded

## Elevation
- Cards: border-based (Notion principle), no heavy shadows
- Maximum: shadow-sm
- Dark hero/footer: flat, no elevation

## Grid
- Max-width: 1280px (container-site)
- Gutters: 4/sm: 16px, 6/lg: 32px
- Section content: 65ch max-width for body text
